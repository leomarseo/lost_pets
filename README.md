### ADD BOOTSTRAP AND SIMPLE_FORM

*terminal*
`yarn add bootstrap`

*application.scss*
`@import "bootstrap/scss/bootstrap"`

*gemfile*
`gem 'simple_form'`

*terminal*
`bundle install`
`rails generate simple_form:install --bootstrap`
